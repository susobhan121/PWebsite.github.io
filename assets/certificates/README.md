# Certificates Folder

Place your scanned or photographed certificates here with EXACTLY these filenames:

- `ai4a.jpg` → AI4A (Artificial Intelligence for All) certificate
- `design-thinking.jpg` → Design Thinking project certificate
- `ace.jpg` → ACE project certificate
- `coursera.jpg` → Coursera online course certificate
- `netcad.jpg` → NetCAD networking certificate
- `matlab.jpg` → MATLAB achievement/certificate
- `yuva-ai.jpg` → Yuva AI initiative certificate

**Tip**: Scan at 150–300 DPI, save as JPG. Keep each under 500KB for fast loading.
